Конфигурационный компилятор (Вариант 28)

Инструмент командной строки для трансляции учебного конфигурационного языка в TOML формат.

Описание задания

Разработан компилятор для учебного конфигурационного языка согласно варианту №28:
- Чтение конфигурации из стандартного ввода (stdin)
- Вывод результата в стандартный вывод (stdout) в формате TOML
- Обнаружение и вывод синтаксических ошибок

 Структура проекта
 config_project/
├── config_compiler_v28.py # Основной компилятор
├── test_config_compiler_v28.py # Модуль тестирования

├── database_config.txt # Пример 3: Конфигурация БД
└── README.md # Этот файл
